## 引用版本
* React 16.8.6
* Webpack 4.30.0
* Babel 7.4.3
* express  4.16.1
## 下载
* `git clone https://gitee.com/huayu611/webpack-express-react-demo.git`
* `cd webpack-express-react-demo`
##开发配置
* `yarn install / npm install`
* `yarn start / npm start`
##介绍
* Webpack，express和React集成。一个简单的demo。可以用这个Demo练习，也可以用来搭建初始框架。
